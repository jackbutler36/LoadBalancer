# LoadBalancer
CSCE 412 project 2

### How to compile code:

g++ loadbalancer.h *.cpp

### How to run code

./a.out

This uses default number of webservers (10) and total time to run (10000)

./a.out 15 12000

This method allows user input for number of webservers (15) and total time to run (12000)
